🛒 Carrinho de Compras

Desenvolvi este projeto com o objetivo de praticar lógica de programação e manipulação do DOM utilizando JavaScript puro. A aplicação simula um carrinho de compras interativo, onde o usuário pode selecionar produtos, definir quantidades e visualizar automaticamente os itens adicionados junto ao valor total atualizado em tempo real.

Implementei uma lógica que realiza o cálculo dinâmico dos preços, tratando os dados diretamente da interface e atualizando o carrinho sem recarregar a página. Além disso, desenvolvi funcionalidades para limpeza do carrinho e controle do estado da aplicação, garantindo uma experiência mais fluida e organizada para o usuário.

O projeto também reforça conceitos importantes como estruturação de código, separação de responsabilidades e interação entre as camadas de interface, estilo e lógica.

🚀 Destaques técnicos:
- Manipulação dinâmica do DOM
- Atualização em tempo real da interface
- Cálculo automático de valores com base na quantidade
- Estruturação do código em funções reutilizáveis
- Controle de estado da aplicação (itens e total do carrinho)
- Interação com eventos de clique

🛠️ Tecnologias utilizadas:
- HTML5
- CSS3 (Grid e estilização moderna)
- JavaScript (ES6+)
- Google Fonts
